    Caption Application¶ ↑

    This application will be designed to let users post image files. User will be able to add captions to images and vote on the best caption.

    Using ruby version 2.2.1

    with:

    • [respec](github.com/rspec/rspec-rails) with [capybara](github.com/jnicklas/capybara) for testing.

    • [Devise](github.com/plataformatec/devise) for authentication

    • [Mysql2](www.mysql.com/) for the database

    • [Paperclip](github.com/thoughtbot/paperclip) for handling image uploads.

    • May consider using an emberjs front end eventually.
